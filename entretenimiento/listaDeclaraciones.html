Listas de Declaraciones en CSS

Organizando Estilos de Forma Eficiente

Una lista de declaraciones en CSS es el conjunto de reglas de estilo que se aplican a un elemento. Cada declaración consiste en una propiedad y un valor, separados por dos puntos : y terminados con un punto y coma ;. Estas listas permiten controlar de manera precisa el aspecto de cada elemento HTML.

Estructura de una Lista de Declaraciones

Las listas de declaraciones siguen una estructura sencilla pero esencial en CSS:

/* Selector de etiqueta */
p {
    color: #6c5ce7;
    font-size: 16px;
    margin: 10px 0;
}

En el ejemplo anterior, color, font-size, y margin son propiedades, mientras que #6c5ce7, 16px, y 10px 0 son sus valores. Juntos forman una lista de declaraciones para el selector p.

Características de las Declaraciones

Las listas de declaraciones deben cumplir ciertas características para funcionar correctamente:

  • Cada propiedad debe ir seguida de dos puntos : y de su valor correspondiente.
  • Es importante finalizar cada declaración con un punto y coma ; para evitar errores.
  • Las listas de declaraciones se encierran en llaves { } y se aplican a uno o más selectores.
  • El orden de las declaraciones en una lista puede afectar el resultado visual de los estilos.

Ejemplo Completo

/* Selector de clase */
.titulo {
    font-family: Arial, sans-serif;
    font-weight: bold;
    color: #74b9ff;
    text-align: center;
    padding: 10px;
    border-bottom: 2px solid #6c5ce7;
}

Este bloque define una lista de declaraciones para la clase .titulo, aplicando estilos de color, tipografía y alineación al elemento que utilice esta clase.
