Uso de <hgroup> en HTML

Organización Jerárquica de Encabezados

La etiqueta <hgroup> en HTML se utiliza para agrupar varios encabezados relacionados jerárquicamente, indicando que forman una unidad. Es especialmente útil cuando un título y subtítulo representan una sección o tema en común.

¿Cuándo Usar <hgroup>?

La etiqueta <hgroup> debe usarse cuando se quiere especificar que un conjunto de encabezados pertenecen a un mismo bloque de contenido. Aquí algunos casos comunes:

  • Cuando un título principal tiene subtítulos de soporte.
  • Para mantener la semántica y accesibilidad del contenido.
  • En artículos, entradas de blogs, y secciones con múltiples encabezados.

Ejemplo Básico

<hgroup>
    <h1>Guía Completa de HTML</h1>
    <h2>Todo lo que necesitas saber sobre HTML5</h2>
</hgroup>

En este ejemplo, el uso de <hgroup> indica que el título principal "Guía Completa de HTML" y su subtítulo "Todo lo que necesitas saber sobre HTML5" están relacionados y forman un solo encabezado temático.

Ejemplo Complejo

<hgroup>
    <h1>Desarrollo Web Moderno</h1>
    <h2>Tecnologías, herramientas y mejores prácticas</h2>
    <h3>Guía para 2024</h3>
</hgroup>

En este ejemplo más complejo, el <hgroup> agrupa un título principal y dos subtítulos adicionales, todos relacionados con el tema de "Desarrollo Web Moderno". Esto mejora la organización y estructura de la página.
