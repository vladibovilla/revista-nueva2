CSS: Hojas de Estilo en Cascada

Domina el Estilo de tu Sitio Web

CSS (Cascading Style Sheets) es el lenguaje que usamos para dar estilo y diseño a nuestras páginas web. A través de CSS, podemos cambiar los colores, fuentes, posicionamiento y efectos visuales de nuestros elementos HTML.

Componentes Principales de CSS

1. Selectores

Los selectores en CSS indican los elementos HTML a los que queremos aplicar estilos. Algunos ejemplos comunes de selectores son:

/* Selector de clase */

.miClase { color: blue; }

/* Selector de ID */

#miID { background-color: yellow; }

/* Selector de tipo */

p { font-size: 16px; }

2. Propiedades y Valores

Una propiedad es el aspecto que queremos modificar (por ejemplo, `color` o `font-size`), y el valor es la configuración de esa propiedad. Por ejemplo:

color: red;

font-size: 20px;

background-color: #f1c40f;

3. Declaraciones y Reglas

Una declaración CSS incluye una propiedad y un valor (por ejemplo, `color: red;`). Varias declaraciones agrupadas para un selector forman una regla. Así es como se define un estilo en CSS:

h1 {

color: #2980b9;

font-size: 2.5em;

}

4. Comentarios

Los comentarios se usan para hacer notas en el código CSS y no se aplican al estilo de la página. Los comentarios en CSS se escriben así:

/* Este es un comentario CSS */

5. Métodos de Aplicación de CSS

Existen tres formas principales de aplicar CSS en un documento HTML:

  • CSS en línea: se aplica directamente en la etiqueta HTML, con el atributo `style`.
  • CSS externo: se vincula a un archivo CSS separado.

Al entender estos componentes, puedes crear estilos potentes y personalizados para tus sitios web. ¡Explora y juega con los diferentes selectores y propiedades para ver los resultados en tus diseños!
